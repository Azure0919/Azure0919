nginx openresty content_by_lua_file 404

配置好lua脚本如下错误:

  • nginx openresty content_by_lua_file 404错误:2020/02/10 19:18:59 [error] 21231#0: *63 failed to load external Lua file “/home/l/lua/content_lua.lua”: cannot open /home/l/lua/content_lua.lua: Permission denied, client: [ip], server: localhost, request: “POST /lua_server HTTP/1.1”, host: “[ip]+[port]”

nginx配置文件里添加

user root root;

即可
